# Zendesk Best Search

A Zendesk navbar app which allows you to search for macros and triggers using the Zendesk API.

### Filter by:

* Tag
* Comment (macros only)
* Notifications (triggers only)
* Created date
* Updated date

Please submit bug reports to Issues. Pull requests are welcome.
